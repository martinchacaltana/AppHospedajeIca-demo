## Descripción
Aplicación web full-stack para la gestión completa de hospedajes turísticos en Ica. Permite a los usuarios buscar, reservar y administrar habitaciones, mientras el administrador controla clientes, reservas y reportes.

## Características principales
- Registro y login de usuarios
- Búsqueda de hospedajes por fecha y tipo
- Reservas en tiempo real
- Panel de administración
- Generación del JAR ejecutable
- Responsive (funciona en celular y PC)
